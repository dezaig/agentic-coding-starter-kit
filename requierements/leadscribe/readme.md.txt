# LeadScribe — Générateur d’emails de vente personnalisés

## Aperçu
LeadScribe créé des séquences d’email hyper personnalisées (à partir de profils/URLs), et suit les réponses.

## Utilisation
1. Import/collez vos prospects.
2. Choisissez un template de séquence.
3. Génération IA + personnalisation champs clés.
4. Envoi via Gmail/Outlook API, suivez opens/clics/replies.

## Cible
SDR/AE, consultants, agences growth.

## Fonctionnalités clés
- Séquences, A/B test
- Envoi via OAuth (boîte de l’utilisateur)
- Tracking, boîte de réception
- Analytics par campagne

## Modèle économique & rentabilité
- Abonnements par siège + volume d’envoi.
- **Très rentable** (COGS faibles, ROI direct pour clients).

## Coûts (COGS)
- LLM texte (faible)
- API mail/IMAP (faible)
**Estimation**: faible.

## Garde-fous coûts
- Caps journaliers d’envoi
- Conformité RGPD/opt-out automatique

## Risques
- Délivrabilité
- Conformité anti-spam

## KPI
