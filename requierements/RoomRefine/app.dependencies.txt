# RoomRefine - Project Dependencies (Windows + npm)

[REQUIRED]
- FFmpeg:
  winget install --id Gyan.FFmpeg -e
  ffmpeg -version
- PostgreSQL:
  POSTGRES_URL=...sslmode=require (remote) OR local Postgres 16+
- Storage (R2/S3)

[OPTIONAL]
- Redis (jobs)
- Stripe CLI
- Computer vision/ML providers:
  * Segmentation/Depth/Normal maps API OR self-hosted models later

[COMMANDS]
npm ci
npm run db:push
npm run dev
