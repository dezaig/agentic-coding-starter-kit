# ClauseGuard — Audit de contrats par IA

## Aperçu
ClauseGuard analyse les contrats, identifie les clauses à risque (responsabilité, paiement, IP) et propose des reformulations claires.

## Utilisation
1. Déposez un PDF/DOCX.
2. L’IA segmente, classe, score les clauses.
3. Consultez le rapport, exportez les recommandations.

## Cible
TPE/PME, indépendants, services juridiques.

## Fonctionnalités clés
- Parsing doc + classification clauses
- Résumés en langage clair
- Suggestions d’édition
- Dates clés et rappels

## Modèle économique & rentabilité
- Prix par document + abonnements pour cabinets/PME.
- Valeur perçue élevée → **bonne rentabilité** si pricing premium.

## Coûts (COGS)
- LLM plus intensif par doc (moyen)
- Stockage docs (faible)
**Estimation**: moyen (pilotable par quotas).

## Garde-fous coûts
- Aperçu gratuit partiel (page/chapitres limités)
- Facturation avant analyse complète
- Batch + cache pour sections récurrentes

## Risques
- Variabilité qualité OCR/parse
- Responsabilité perçue → disclaimers

## KPI
- Coût IA/document
- Taux de conversion essai → achat
- Temps de traitement
- NPS pro
